# NodeLook
NodeLook is a general website checker for users facing internet restrictions. It helps users determine whether a service is accessible through their current connection, or whether it is filtered or unavailable.

Some of the services this app checks, and it may be useful for users in countries where internet restrictions exist, include:

* Telegram — China, Russia, Turkmenistan, Pakistan, Iran
* Wikipedia — China, Turkey, Myanmar, Pakistan, Iran
* Internet Archive — China, Russia, Iran
* Dropbox — China, Iran
* GitHub — China, Iran
* Google — China, Iran
