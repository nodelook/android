# Changelog

All notable changes to this project will be documented in this file.

## <a id="v1.5.0"></a>[v1.5.0] - 2026-07-19

### Added

- Added Android TV support.
- Added a power connection status checker.
- Updated the list of connectivity test endpoints.

### Changed

- Changed the **Stop** button behavior to stop the ongoing test instead of closing the app.

### Fixed

- Fixed a crash when running ping tests on older Android versions.

## <a id="v1.4.0"></a>[v1.4.0] - 2026-07-06

Migrated to Kotlin, added IP version 6 ping support, and improved landscape mode

## <a id="v1.3"></a>[v1.3] - 2026-06-26
